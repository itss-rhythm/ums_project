SPECIFICS OF THE PROJECT


This project was made keeping in mind the general way APACHE TOMCAT 8.5 requires to execute any project. Connectivity between HTML and SERVLET pages is given by the usual "web.xml" file . Some context parameters are used to give name of the databse used and for the password of the DBMS therefore used. Before running the project one must do the following........
1.open "web.xml" and input the password of your MySql in "param-value" tag
2.in "web.xml" and input the name of your database in "param-value" tag
3. in sql run the following command
CREATE TABLE `stdlog` ( `roll` VARCHAR(45) NOT NULL, `password` VARCHAR(45) NOT NULL, `phone` VARCHAR(45) NOT NULL, `address` VARCHAR(45) NULL, `email` VARCHAR(45) NULL, PRIMARY KEY (`roll`));
4.connector provided will be in .doc format (as .jar file cant be sent through mail) so neens to be converted to .jar file
